# UPNP-manager

## Guide rapide
- Pour lancer : `python main.py`
- Pour déployer : voir le Dockerfile
- Pour lancer l’API REST : `python api.py`
- Pour tester : `pytest`

## Exemples de configuration
```json
{
  "port": 8080,
  "upnp_enabled": false,
  "auto_start": false,
  "theme": "light",
  "max_file_size": 50,
  "allowed_extensions": [
    ".html", ".css", ".js", ".png", ".jpg", ".jpeg", ".gif", ".ico"
  ]
}
```
